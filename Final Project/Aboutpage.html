Why This Project?

Music has been a huge factor in my life for as long as I can remember. Although I am no longer studying music in college, I still partcipate in my university's bands regularly and practice on a daily basis. I enjoy performig and listening to music. As I developed this website, I wanted to share some of my favorite trombone tunes that I listen to regularly. Music is an essential part of our lives, and many of us rarely have time to sit down and enjoy the beauty of it. Alas, I found a way to bring the creativity and blessing of music to my web programming class.

Technology I Used

This website was designed using HTML, CSS, and PHP. The youtube video and spotify album was imbedded into this website using the HTML iframe tag. Each page in the website corrosponds to a different HTML and CSS file to make the website very readable and easy to edit. PHPMyAdmin was used to control the sign-in system by sending and retrieving data from a database.

What I Learned

This class is my first hands-on experience with web programming. Through this class, I learned several programming languages, including HTML, CSS, Javascript, and PHP. I also learned how front-end programming connects to back-end programming, which will provide much utility in the future. This class was very enjoyable and I am able to use this website as a display of my programming skills.

What is Trombonium

Trombonium is a website designed to share trombone videos with musicians and non-musicians alike. The archive and homepage make viewing trombone videos easier than ever.

What do I get from Trombonium?

Trombonium is a way for me to share my passion of trombone with the rest of the world. You can decide whether you enjoy listening to these videos. However, I urge you to give this website a chance and listen to videos from a variety of different genres.

How can Trombonium help me?

Music is a cure for the soul and can reach many parts of the brain that we normally don't use. The power of music can help us express our emotions when all words are lost. Even better, music is also enjoyable. Take some time to escape from your daily problems and relax while enjoying many different trombone videos.

What can I do?

Users of this website can view various YouTube and Spotify links on the archive page. The homepage always shows the latest releases so users can stay in touch with the trombone community. A contact page is also avaliable for users to submit various questions, comments, or inquiries.

Checklist

  • Database Usage - Several databases are used to store sign-in info and the various YouTube and Spotify links.
  • Ajax Usage - Ajax is used in the admin page to update the links in the archive
  • Theme - Each page has the same navigation bar, image, and footer. The light background and colors help make the website pleasing to the eye.
  • New Library Usage - The HTML iframe tag was used to implement the latest links directly onto the homepage
  • Javascript Usage - Javascript is used in all pages with a form to update the page if the data was submitted or if a field is missing
  • Membership Area - This website has a working membership page where admin can access the archives and edit them
  • General User - Although user's cannot register for the sign-in page, there is a working username and password. For security purposes, I will not be sharing the login information on here.
